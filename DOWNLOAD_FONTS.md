# Download Noto Nastaliq Urdu Fonts

## Quick Download Steps

1. **Visit Google Fonts:**
   - Go to: https://fonts.google.com/noto/specimen/Noto+Nastaliq+Urdu

2. **Download the font family:**
   - Click the **"Download family"** button (top right)
   - This will download a ZIP file

3. **Extract the ZIP:**
   - Extract the downloaded ZIP file
   - You'll find TTF font files inside

4. **Copy font files:**
   - Copy these 4 files to: `public/assets/fonts/nastaliq/`
     - `NotoNastaliqUrdu-Regular.ttf`
     - `NotoNastaliqUrdu-Medium.ttf`
     - `NotoNastaliqUrdu-SemiBold.ttf`
     - `NotoNastaliqUrdu-Bold.ttf`

5. **Verify the structure:**
   ```
   public/
     assets/
       fonts/
         nastaliq/
           NotoNastaliqUrdu-Regular.ttf
           NotoNastaliqUrdu-Medium.ttf
           NotoNastaliqUrdu-SemiBold.ttf
           NotoNastaliqUrdu-Bold.ttf
   ```

6. **Restart your dev server:**
   ```bash
   npm run dev
   ```

## Alternative: Direct Download Links

If the Google Fonts page doesn't work, you can try these direct links:

- Regular: https://github.com/google/fonts/raw/main/ofl/notonastaliqurdu/NotoNastaliqUrdu-Regular.ttf
- Medium: https://github.com/google/fonts/raw/main/ofl/notonastaliqurdu/NotoNastaliqUrdu-Medium.ttf
- SemiBold: https://github.com/google/fonts/raw/main/ofl/notonastaliqurdu/NotoNastaliqUrdu-SemiBold.ttf
- Bold: https://github.com/google/fonts/raw/main/ofl/notonastaliqurdu/NotoNastaliqUrdu-Bold.ttf

Right-click each link and "Save As" to download, then place in `public/assets/fonts/nastaliq/`

## After Downloading

Once the files are in place, the font will work automatically. No code changes needed!
